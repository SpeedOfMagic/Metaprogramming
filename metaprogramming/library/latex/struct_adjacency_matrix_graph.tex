\doxysubsection{Detailed Description}
\subsubsection*{template$<$class vertexes, class matrix$>$\newline
struct Adjacency\+Matrix\+Graph$<$ vertexes, matrix $>$}

Represents graph as an adjacency matrix. Element in row i, column j must be \mbox{\hyperlink{struct_objects_1_1_boolean}{Objects\+::\+Boolean}}. If it\textquotesingle{}s Boolean$<$false$>$, then there\textquotesingle{}s no edge between them. Otherwise there is. If element is Boolean$<$true$>$, then the weight is \mbox{\hyperlink{struct_null_type}{Null\+Type}}. Otherwise it\textquotesingle{}s equal to this element. If you wish to have Boolean as a weight, then consider wrapping it in some other class. \begin{DoxySeeAlso}{See also}
