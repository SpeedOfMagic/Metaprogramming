\subsubsection*{template$<$typename T$>$\newline
class Stream$<$ T $>$}

Represents an analogue of std\+::ranges. Unlike std\+::ranges, this class is compatible with earlier standards. It was made specifically for indexes of vertexes, but it can be used for other purposes.
